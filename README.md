# CalculaIMC

Este projeto é o resultado da aula introdutória da discplina de Android Básico da especialização em programação para dispositivos móveis, com foco na plataforma Android.

## Tópicos Abordados

*   **Android Studio:** Apresentação da IDE de desenvolvimento oficial do Android.
*   **Componentes Visuais:** Utilização de componentes de interface como `TextView`, `EditText` e `Button` para construir a tela do aplicativo.
*   **Modelo de Eventos:** Manipulação de interações do usuário, como cliques em botões (`setOnClickListener` e `setOnLongClickListener`).
*   **Activities:** Conceito de telas no Android e o ciclo de vida de uma `Activity` (demonstrado através do método `onCreate`).
*   **Técnicas de Depuração:** Utilização de `Toast` para exibir mensagens rápidas e a validação de campos com `EditText.error`.

## Sobre o Projeto

O aplicativo "Calcula IMC" é uma calculadora simples para o Índice de Massa Corporal. O usuário insere seu peso e altura, e o aplicativo calcula e exibe o resultado.

A interface foi construída utilizando `LinearLayout` para organizar os componentes. A lógica do cálculo e a interação com os botões estão implementadas na `MainActivity`, utilizando a linguagem Kotlin.
